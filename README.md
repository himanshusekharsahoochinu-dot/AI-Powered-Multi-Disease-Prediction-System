## 📖 Overview

The **AI-Powered Multi Disease Prediction System** is a Machine Learning-based healthcare application developed using **Python** and **Streamlit**. It predicts the risk of multiple diseases using patient health parameters and generates a downloadable PDF health report.

The application currently supports prediction of:

🩸 Diabetes
❤️ Heart Disease
🩺 Kidney Disease
🧬 Liver Disease

✨ Features
Multi-disease prediction using Machine Learning
Patient information management
Interactive healthcare dashboard
Confidence score for predictions
Health recommendations
PDF report generation
Clean and user-friendly interface...
